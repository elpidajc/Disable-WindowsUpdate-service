# Disable-WindowsUpdate-service
Disable WindowsUpdate service by deleting key catche of windows-update-service
# what and how?
what:
  There are two windows scripts:createfile.bat and recover.bat.
how:
  1.download two files mentioned.
  2.double click createfile.bat with administrator right to disable windows-auto-update service.
# side effect:
  1. if you disable windows update service using my way, you will not download applications in microsoft app store.
# how to recover:
  1.double click recoverfile.bat with administrator right to recover windows-auto-update service.
  2.maybe you need to reboot your computer.
